# dostaffkin

Лаконичный SPA на Angular для оформления и отслеживания заявок на доставку с расчётом маршрута и стоимости через Yandex.Maps.

**Ключевой функционал**
- Обратное геокодирование и автодополнение адресов (Yandex.Maps `ymaps`).
- Построение маршрута и оценка расстояния/времени через `ymaps.multiRouter`.
- Расчёт стоимости по размеру посылки и скорости доставки (настраиваемые тарифы в `order.config.ts`).
- Форма заказа с валидацией и отправкой на бэкенд через `DeliveryApi.createDelivery`.
- Уведомления об ошибках/успехе через `ngx-toastr`.

**Технологический стек**
- Язык: TypeScript
- Фреймворк: Angular 21
- Библиотеки: `rxjs`, `ngx-toastr`, Yandex.Maps (глобальный `ymaps`)
- Инструменты разработки: `@angular/cli`, `vitest` (dev)
- Менеджер пакетов: npm (в проекте указано `npm@10.9.2`)

**Краткая инструкция по установке и запуску**
1. Установить зависимости:

```bash
npm install
```

2. Запустить в режиме разработки (dev server):

```bash
npm start
# Откроется http://localhost:4200
```

3. Сборка для продакшена:

```bash
npm run build
```

4. Полезные команды:

```bash
npm run watch   # rebuild в режиме наблюдения
npm run test    # запуск тестов (vitest)
```

**Требования и примечания**
- В проекте используется глобальный объект `ymaps` — необходимо подключить Yandex.Maps API (скрипт с ключом) в `index.html` или другом месте загрузки скриптов.
- Это фронтенд-проект: для работы отправки заявок нужен бэкенд, который обрабатывает вызовы `DeliveryApi`.
- Проверяйте версии Node/npm совместимые с указанным менеджером пакетов.
